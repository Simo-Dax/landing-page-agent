# Advanced Copywriting — README

## Overview
Framework avanzati per copy ad alta conversione: formule classiche (AIDA, PAS, BAB, FAB), template sezione-per-sezione, voice & tone guidelines, copy editing checklist.

## Quick Start
```bash
# Quando usare:
"Scrivi il copy completo per questa landing page"
"Ho bisogno di copy per la sezione features"
"Applica framework AIDA a questa sales page"

# Output: Copy strutturato sezione per sezione con formule applicate
```

## Formule Copywriting
1. **AIDA**: Attention → Interest → Desire → Action (landing page generica)
2. **PAS**: Problem → Agitate → Solve (pain point forte)
3. **BAB**: Before → After → Bridge (trasformazione)
4. **FAB**: Features → Advantages → Benefits (feature page)
5. **4 U's**: Useful, Urgent, Unique, Ultra-specific (headline check)

## Template per Sezioni
- **Hero**: Headline + Subheadline + CTA + Trust signal
- **Social Proof**: Testimonial specifici con risultati misurabili
- **Features**: Benefit-focused (non feature dump)
- **How It Works**: 3-step semplice
- **FAQ**: Anticipa obiezioni top 5-7
- **Pricing**: 3 piani max, recommended highlighted
- **Final CTA**: Ultimo push con risk reversal

## Tone Guidelines
- Diretto, no fluff (segui `context/brand/tone_of_voice.md`)
- Specifico: numeri, dati, outcome tangibili
- No superlatives vuoti, no jargon inutile

## Integration
Usa dopo Brief + Architettura → Combina con Marketing Psychology per leve → Valida con Self-Check.
